# 📋 ניתוח הערות מאמן מומחה

## 🔍 סיכום ההערות מהשיחה

### **הערה 1: חסר הסבר על מטרת החקירה (S2)**
**מיקום:** במעבר ל-S2 (בקשת אירוע ספציפי)

**מה כתוב:**
```
מאמן: "עכשיו, כדי שנוכל להבין לעומק את המצב,"
הערה: (ולזהות את הדפוס שלך במצבים כאלה!)
```

**מה חסר:** הסבר על **המטרה** - לזהות דפוס חוזר!

**מה צריך להיות:**
```
"עכשיו, כדי שנוכל להבין לעומק את המצב ולזהות את הדפוס שלך במצבים כאלה,
אני מבקש שתיקחי אותי לרגע מסוים..."
```

---

### **הערה 2: חסרה שאלה "מה בדיוק קרה?"**
**מיקום:** בסוף בקשת האירוע ב-S2

**מה כתוב:**
```
מאמן: "ספרי על פעם אחת - עם מי זה היה? מתי זה היה?"
הערה: להוסיף- מה בדיוק קרה שם?
```

**מה צריך להיות:**
```
"ספרי על פעם אחת - עם מי זה היה? מתי זה היה? מה בדיוק קרה שם?"
```

---

### **הערה 3: צריך להבהיר שוב את המטרה (S3)**
**מיקום:** במעבר ל-S3 (רגשות)

**מה כתוב:**
```
מאמן: "עכשיו, אני רוצה יותר להבין איזה רגשות..."
הערה: צריך להבהיר שוב מה אנחנו מחפשים- כדי שנוכל לזהות את הדפוס שלך
```

**מה צריך להיות:**
```
"עכשיו, כדי שנוכל לזהות את הדפוס שלך, אני רוצה להבין איזה רגשות..."
```

### **הערה 4: שגיאת מינוח - "תחושות" במקום "רגשות"**
**מיקום:** בחקירת רגשות (S3)

**מה כתוב:**
```
מאמן: "התחושות האלו של חוסר אמון..."
הערה: (אלו הרגשות!)
```

**הבעיה:** המאמן אומר "תחושות" במקום "רגשות".

**תיקון נדרש:** להשתמש במילה **"רגשות"** בלבד, לא "תחושות"!

---

### **הערה 5: סדר שאלות שגוי - גוף לפני רגש**
**מיקום:** בחקירת רגשות (S3)

**מה כתוב:**
```
מאמן: "איפה הרגשת את התחושות האלו בגוף?"
הערה: אחרי שתענה על ההרגשות (ולא התחושות) אפשר לשאול על הגוף
```

**הבעיה:** המאמן שואל על **מיקום בגוף** לפני ש**זיהה את הרגש עצמו**.

**הסדר הנכון:**
1. קודם: "מה הרגשת?" → קבל שם של רגש (כעס, פחד, עצב...)
2. רק אז: "איפה הרגשת את זה בגוף?"

---

### **הערה 6: חזרה מיותרת על רשימת רגשות**
**מיקום:** אחרי שהמשתמש אמר 4 רגשות

**מה כתוב:**
```
משתמש: "תסכול, כעס, עלבון ועייפות"
מאמן: "תסכול, כעס, עלבון ועייפות... אני שומע שיש הרבה רגשות חזקים באותו רגע."
הערה: מיותר, חוזר על עצמו
```

**הבעיה:** המאמן חוזר על כל הרשימה + מוסיף סיכום מיותר.

**מה צריך להיות:**
```
משתמש: "תסכול, כעס, עלבון ועייפות"
מאמן: "ספרי לי יותר על התסכול הזה - איפה הרגשת אותו?"
```

---

### **⚠️ הערה 7: הבעיה הכי קריטית - דילוג על S6+S7!**
**מיקום:** אחרי S5 (מעשה מצוי + רצוי)

**מה כתוב:**
```
מאמן: "מה עשית?" → "מה רצית לעשות?"
הערה: קודם (לפני "איך היית רוצה…") צריך פה להופיע:
  1. סיכום של הדפוס (רגש אמירה פעולה)
  2. אישוש שלו על ידי שאלות:
     - האם אתה מכיר את עצמך מופיע כך בעוד מקומות?
     - האם זה קורה רק עם...?
     - האם זה תלוי בנסיבות או במציאות?
     - איפה עוד זה קורה?
     - מניין עוד אתה מכיר תגובה זהה שלך?
```

**🚨 הבעיה המבנית:**
המאמן קופץ מ-**S5 (מצוי+רצוי)** ישר ל**סיכום סופי**, בלי לעבור על:
- **S6 (פער)**: "איך תקרא לפער בין מצוי לרצוי?"
- **S7 (דפוס)**: "איפה עוד זה קורה?" + אישוש הדפוס

**הסדר הנכון צריך להיות:**
```
S5: מצוי + רצוי
  ↓
S6: שם לפער + ציון
  ↓
S7: זיהוי דפוס (2-3 דוגמאות + אישוש)
  ↓
S8: רווחים + הפסדים
  ↓
...
```

### **הערה 8: סיכום סופי שגוי - מסכם סיפור במקום דפוס**
**מיקום:** בסיכום הסופי

**מה כתוב:**
```
מאמן: "בואי נסכם את מה שקרה: באותו רגע של שיחה עם בעלך... [סיפור ארוך]"
הערה: במקום לסכם שוב את הסיפור צריך להופיע פה סיכום של הדפוס שנמצא בתוך הסיפור
```

**הבעיה:** המאמן מסכם את **הסיפור** (מה קרה), לא את **הדפוס** (מה חוזר).

**מה צריך להיות:**
```
"בואי נסכם את הדפוס שזיהינו:
כשיש לך רצון שיוצא מהשגרה, ומישהו מטיל ספק בו או שואל הרבה שאלות,
אתה מגיבה ב[פעולה חוזרת] - זה קורה גם עם בעלך וגם ב[דוגמה 2].
המצבים שונים, אבל התגובה שלך זהה. נכון?"
```

## 5. **תיקון מינוח: "רגשות" לא "תחושות"**

  - רגש = כעס, עצב, פחד...
  - תחושה = תחושה פיזית (כבד בחזה, לחץ בראש...)

**מה זה מונע:**
- בלבול בין רגש (emotional) לתחושה (physical sensation)

## 6. **תיקון סדר שאלות: רגש → תיאור → גוף**

### קובץ: `backend/app/bsd_v2/prompt_compact.py`

**מה תוקן:**
- הבהרת הסדר הנכון:
  1. קבל **שם רגש** ("כעס")
  2. חקור את הרגש: "ספר לי יותר על ה-[רגש]"
  3. **רק אז** שאל על מיקום בגוף: "איפה הרגשת את זה?"

**דוגמה:**
```
שם רגש → תיאור הרגש → מיקום בגוף → רגש נוסף
```

**מה זה מונע:**
- שאלה על מיקום בגוף לפני שזיהינו את הרגש עצמו
