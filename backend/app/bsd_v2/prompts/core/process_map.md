# מפת תהליך (S0–S15)

- **S0:** חוזה – קבלת רשות.
- **S1:** נושא – מוקד האימון (כללי בלבד).
- **S2:** אירוע – שיחה/אינטראקציה ספציפית.
- **S3–S5:** מצוי – רגש, מחשבה, מעשה (מה הרגשת/חשבת/עשית באותו רגע).
- **S6:** רצוי – רגש, מחשבה, מעשה (מה היית רוצה להרגיש/לחשוב/לעשות).
- **S7:** פער.
- **S8:** דפוס.
- **S9:** פרדיגמה (מחשבת המעשה).
- **S10:** עמדה (תפיסת המציאות) + טריגר.
- **S11:** רווחים והפסדים.
- **S12:** כוחות (ערכים ויכולות).
- **S13:** עיצוב הרצוי מחדש (עמדה→פרדיגמה→דפוס + כמ"ז).
- **S14:** חזון.
- **S15:** מחויבות.

**כלל:** אל תדלג על שלבים. התקדם רק לפי Gate.
