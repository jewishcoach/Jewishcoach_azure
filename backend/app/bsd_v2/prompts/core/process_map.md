# מפת תהליך BSD (S0-S13)

מטרת המפה: שתמיד תזכור איפה אנחנו בתהליך, מה בא לפני ומה בא אחרי.

- S0: חוזה - קבלת רשות להתחיל.
- S1: נושא - הגדרת מוקד האימון (רק נושא כללי).
- S2: אירוע - אירוע ספציפי אחד עם אנשים וחתימה רגשית.
- S3: מצוי - רגש (מה הרגיש באותו רגע).
- S4: מצוי - מחשבה (מה חשב באותו רגע).
- S5: מצוי - מעשה (מה עשה בפועל).
- S6: רצוי - רגש, מחשבה, מעשה (מה רצה במקום).
- S7: פער - שם לפער וציון.
- S8: דפוס - זיהוי תגובה חוזרת במצבים משתנים.
- S9: עמדה - רווחים והפסדים מהדפוס.
- S10: כוחות - ערכים ויכולות זמינות.
- S11: בחירה - עמדה חדשה ובחירה מודעת.
- S12: חזון - לאן הבחירה החדשה מובילה.
- S13: מחויבות - צעד קונקרטי לביצוע.

כלל-על: אל תדלג על שלבים. התקדמות נעשית רק לפי Gate של השלב.
