# מפת תהליך BSD (S0-S12)

מטרת המפה: שתמיד תזכור איפה אנחנו בתהליך, מה בא לפני ומה בא אחרי.

- S0: חוזה - קבלת רשות להתחיל.
- S1: נושא - הגדרת מוקד האימון (רק נושא כללי).
- S2: אירוע - אירוע ספציפי אחד עם אנשים וחתימה רגשית.
- S3: רגשות - כניסה חווייתית לעומק מה שהרגיש באותו רגע.
- S4: מחשבה - המשפט המילולי שעבר בראש באותו רגע.
- S5: מעשה+רצוי - מה עשה בפועל, ומה רצה במקום (מעשה/רגש/מחשבה).
- S6: פער - שם לפער וציון.
- S7: דפוס - זיהוי תגובה חוזרת במצבים משתנים.
- S8: עמדה - רווחים והפסדים מהדפוס.
- S9: כוחות - ערכים ויכולות זמינות.
- S10: בחירה - עמדה חדשה ובחירה מודעת.
- S11: חזון - לאן הבחירה החדשה מובילה.
- S12: מחויבות - צעד קונקרטי לביצוע.

כלל-על: אל תדלג על שלבים. התקדמות נעשית רק לפי Gate של השלב.
