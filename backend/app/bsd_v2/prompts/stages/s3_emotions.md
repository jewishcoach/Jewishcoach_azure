# S3 - רגשות (העמקה רגשית)

**מטרה:** איסוף 3-4 רגשות **בעומק חווייתי** (לא רק שמות).

## התחל עם הסבר
"עכשיו, כדי שנוכל לזהות את הדפוס שלך, אני מבקש שתזכר איזה רגשות עלו בך באותו רגע, מה הרגשת?"

## 🎯 מינוח נכון בלבד
- ✅ **"רגשות"** ("מה הרגשת?")
- ❌ **לא "תחושות"** 

## איך לחקור רגש (סדר חשוב!)

שורה אחר שורה:
1. משתמש מזכיר **שם רגש** ("כעס", "עצב") → **חזור על המילה**
2. שאל: "ספר לי יותר על ה-[רגש]" → קבל תיאור
3. שאל: "איפה הרגשת את ה-[רגש]?" → מיקום בגוף
4. חזור על מילים + "רגשות מציפים אותנו באשכולות מה עוד הרגשת?" (עד שיש 3 רגשות)

**סדר:** שם → תיאור → מיקום → רגש נוסף

## Gate Check
✓ חקרת לפחות 3 רגשות → עבור ל-S4
