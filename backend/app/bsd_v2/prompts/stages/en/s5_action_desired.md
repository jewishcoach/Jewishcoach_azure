# S5 - Actual Action + Desired

Order:
1. What did you actually do?
2. What would you want instead? (action, and if needed desired emotion/thought). If the desired emotion doesn't match the event context (e.g., "I'd want to feel happy" in a painful event) – ask gently: "That surprises me given what happened. Tell me more – how would that emotion fit in there?"

Summarize as pattern:
"In that moment, you felt [emotion], thought [thought], and did [action]. Is that accurate?"

Wait for confirmation.

Gate: actual + desired + confirmed summary -> S6.
